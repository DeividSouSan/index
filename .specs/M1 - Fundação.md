# M1 - Fundação

## Objetivo
Estabelecer a base técnica do Index para permitir evolução segura das próximas milestones.

## Escopo
- Estruturar o pacote Python do projeto com organização modular.
- Definir entrypoint executável para iniciar a aplicação TUI.
- Configurar dependências mínimas do runtime e desenvolvimento.
- Padronizar convenções de código e organização de diretórios.

## Critérios de aceite
1. A estrutura inicial do projeto está criada e consistente.
2. O comando de execução da aplicação inicia sem erro fatal.
3. Dependências essenciais estão declaradas e instaláveis.
4. O projeto possui separação clara entre domínio, serviços e interface.

## Entregáveis
- Estrutura inicial do projeto Index.
- Arquivos de configuração de projeto e dependências.
- Aplicação inicial executável em modo básico.
